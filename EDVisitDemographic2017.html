ED Visit Demographic 2017¶

Below is data exploration exercise of Overlake Emergency Department Visit for 2017. Data set used is from Epic Caboodle Data Warehouse. We are looking at several dimensions of the data such as gender, race, insurance class, discharge type and arrival method.

Data Set from Epic DW¶

In [40]:
edVist <- sqlQuery(dataConn,"select edp.ArrivalMethod, edp.EdDisposition, edp.DischargeDisposition,edp.AcuityLevel,
  edp.FinancialClass, pt.name,pt.Sex, pt.PreferredLanguage, pt.FirstRace, pt.City , pt.PrimaryFinancialClass,
  MONTH(arrivalinstant) monthadm, year(getdate()) - year(birthdate) Age
  from EdVisitFact ed
  join EdVisitProfileDim edp on ed.EdVisitProfileKey = edp.Edvisitprofilekey
  join PatientDim pt on ed.PatientKey=pt.PatientKey 
  where year(ArrivalInstant)=2017")
